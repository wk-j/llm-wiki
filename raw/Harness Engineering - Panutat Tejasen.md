ได้อ่านความเห็นของ อ. หลายท่านว่าควรสอนให้นักศึกษาสามารถ "ตรวจงาน" และ "แนะนำ" AI Agent ให้เขียนโปรแกรมและเลือกใช้ Framework ต่างๆได้อย่าง "ถูกต้อง"
จากประสบการณ์ที่ใช้ AI Agent เขียนโปรแกรมมาจริงๆ บอกได้เลยครับว่า Framework หรือ Library ที่มันเลือกใช้แต่ละตัว มันเกินความสามารถที่เราจะไปแนะนำครับ คือมันรู้มากกว่าเรามาก ถ้าเราคิดทำเช่นนั้น ความเร็วที่เราหวังว่า AI จะช่วยให้ x10 มันจะเป็น /100 แทน ติดที่เรานี่แหละ
สิ่งที่ควรทำคือ สอน "Harness Engineering" ครับ คือ ควบคุมมันด้วยการออกแบบ AI Agent อีกหลายตัวที่ให้มัน Review Code, ตรวจสอบ, ทำ Test Cases, และ End-to-End Testing รวมถึง Security Audit ให้ได้ดี นั่นคือสิ่งที่มนุษย์ทำได้ในเวลานี้ ไม่มีทางครับที่เราจะแนะนำมัน มันไปไกลกว่านั้นมากแล้ว
