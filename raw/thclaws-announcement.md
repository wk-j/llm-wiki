# Panutat Tejasen - thClaws Announcement

Date: April 26, 2026 (approximate, based on "1d" from current date)
Author: Panutat Tejasen
Source: Facebook/Social Media

ขอมาเล่าเรื่องการทำ thClaws ให้เพื่อนๆฟังกันตรงนี้นะครับ สามปีกว่าที่ผ่านมาเราผ่านเข้าสู่โลกของ AI กันชนิดหายใจไม่ทัน เพื่อนๆหลายคนคงแปลกใจว่าทำไมทีมเราทำ thClaws ขึ้นมา...บ้าไปหรือป่าว
เราทำ AI ในระดับ Application มาตั้งแต่ที่ OpenAI เปิด API ให้ใช้ก่อนเปิดตัว ChatGPT และก็ถูกกระแนะกระแหนมาตลอดว่าเป็น LLM Wrapper ซึ่งกระแสนี้เป็นกันทั้งโลก เพราะโลกให้ความสำคัญที่โมเดล
มาถึงตอนที่ Claude Code ออกมา ทีม Anthropic เล่าว่า Sonnet/Opus 3.5 ออกมาก่อนหน้าประมาณ 6 เดือน แต่ไม่มีใครรู้ว่ามันเขียน code ได้ดี จนลองให้มันเข้า File System ได้ แล้วพบว่ามันอ่านไฟล์ source code แล้วทำความเข้าใจได้ หาความสัมพันธ์ระหว่าง code ได้ และช่วยแก้ไข code ได้ด้วย เมื่อใส่ tools ให้มันจัดการไฟล์ได้
ปรากฎการณ์นี้เรียกว่า product overhang คือโมเดลมันทำได้อยู่แล้ว แต่ไม่มี interface ให้มันทำ ก็เลยไม่มีใครรู้ว่ามันทำได้
ก่อนหน้านั้นก็มีคำจำกัดความมาก่อนแล้วว่า AI Agent คือ LLM + Tools + Knowledge แต่ไม่มีใครให้มันมาป้วนเปี้ยนในเครื่องเรา ด้วยเหตุผลความปลอดภัยจากความหลอนของ AI (แน่นอนว่าถึงวันนี้มันก็ยังลบไฟล์ผมอยู่โดยไม่ได้สั่ง)
หลังจากเราพบว่า LLM ทำได้มากกว่าที่เราคิด เราก็เข้าสู่โลกของ Agentic AI กันในระดับบ้าคลั่ง Claude Code ถือเป็นตัวหนึ่งที่สุดยอด โมเดลก็ดี แถมโปรแกรมมันก็ดี และบรรดานักพัฒนาก็สงสัยมาตลอดว่าทำไมมันดี
แล้วฟ้าก็ประทานให้ source code มันหลุดออกมา โลกแทบแตก สามสัปดาห์กว่าๆที่ผ่านผม ผมแทบไม่ได้นอน นั่งดู source code และอ่านงานวิจัยเกี่ยวกับ source code ชุดนี้กันแทบไม่หวาดไม่ไหว (ใช่ครับ มันเป็น source code ที่มีคนทำวิจัยว่ามันทำงานยังไง)
งานที่เกี่ยวกับการคุม LLM จริงๆใน sourse code มีแค่ 1 เปอร์เซ็นต์กว่าๆ ที่เหลือคืองานรอบข้าง เช่น Agentic Loop, Tool routing, Sub-agent dispatch, Sandbox, Context compaction, Skill with script, MCP, Plugins, และที่เราชอบสุดคือ Agent Teams
ยิ่งศึกษาก็ยิ่งหลงไหล source code 500,000+ กว่าบรรทัดของ AI product ที่ถือกันว่าดีที่สุดในโลกอยู่ในมือเรา ไม่ศึกษา ไม่พอร์ต ก็นอนไม่หลับแน่ครับ แล้วก็ไม่ได้หลับ ไม่ได้นอนกัน
ทำ Clean Room Port เป็น Rust ขนาดประมาณ 36,000 บรรทัด เล็กมากเมื่อเทียบกับต้นตอ แต่เราต้องการเฉพาะส่วนสำคัญที่ทำหน้าที่ Harness คือเป็นบังเหียร ที่จะควลคุมเจ้าม้าป่า คือ LLM
Harness Engineering กำลังเป็นเรื่องสำคัญในอนาคต ตัวโมเดลมันก็คงแข่งกันจนเป็น AGI, ASI หรืออะไรต่างๆในอนาคต ถ้าเราจะใช้มันได้ ก็ต้องหัดควบคุมมันตอนที่พอจะควบคุมได้นี่แหละครับ
thClaws เป็นโครงการที่เราทำเพื่อศึกษาการทำ AI Harness คือการสร้างโปรแกรมมาควบคุมให้ AI ทำงานตามที่เราต้องการ
เพื่ออะไร? ตอนนี้เราผ่านยุค LLM Wrapper มาแล้ว Anthropic เพิ่งออก Claude Design ออกมา และนี่แหละคือตัวอย่างของ Wrapper รุ่นใหม่ที่เราเรียกกันว่า Harness หรือ Agentic Application ที่ทำให้หุ้นของบริษัทโปรแกรมหลายตัวร่วงเป็นแถว
จุดหมายต่อไปของเราคือการสร้าง AI Harness หรือ Agentic Application และจะทำได้ดีก็ต้องรู้ว่า Harness ระดับโลกเค้าทำกันยังไง และนั่นคือเหตุผลครับที่เกิด thClaws ขึ้นมา
และเราก็เปิดเป็น Open source (Apache + MIT Dual license เหมือน Rust App ทั่วไป) ชวนเพื่อนๆมาศึกษาและขยายความสามารถของมันต่อไปด้วยกันครับ
