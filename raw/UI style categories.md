Here's a live showcase of all 4 styles side by side:

**① Diegetic HUD** — Iron Man style. The UI *is* the interface — arc meters, scan lines, bracket targeting, real-time bar graphs. Cyan = normal, amber = caution, red = critical.

**② Holographic UI** — Minority Report style. Semi-transparent panels with floating particles, corner brackets, and a glowing line chart. Everything feels like projected light.

**③ Brutalist Futurism** — No decoration, only raw data. Heavy typography, sharp borders, red accents, blinking status indicators. Confrontational by design.

**④ Retro-Futurism** — 80s sci-fi computer aesthetic. CRT scan lines, VT323 pixel font, neon pink/purple/green, blinking cursor. Feels like a starship console from a classic space opera.
