## What Is Soviet Cosmism?

Soviet Cosmism (Русский Космизм) emerged in the late 19th century through thinkers like **Nikolai Fedorov**, **Konstantin Tsiolkovsky**, and **Vladimir Vernadsky**, later intersecting with Soviet space propaganda. It is not mere retrofuturism — it is a *genuinely utopian philosophy* that treated space as sacred ground for human transcendence.

| Pillar | Concept | Visual Expression |
| --- | --- | --- |
| **Космос (Cosmos)** | Space as the ultimate frontier and spiritual realm | Deep star fields, orbital diagrams, celestial mechanics |
| **Воскрешение (Resurrection)** | Technology will defeat death and resurrect all humans | Biological data streams, cellular patterns, genealogy trees |
| **Ноосфера (Noosphere)** | Earth's thinking layer — collective consciousness | Network webs, radiating brainwave patterns |
| **Конструктивизм (Constructivism)** | Form follows function, geometry serves ideology | Bold diagonal lines, stark geometric shapes |
| **Прогресс (Progress)** | Relentless optimism toward the future | Upward arrows, launching rockets, ascending spirals |

### Typography

Soviet Cosmism UI uses a **strict typographic hierarchy** of four roles:

| Role | Font Stack | Use Case |
| --- | --- | --- |
| **Cyrillic Display** | `"Oswald", "Anton", sans-serif` (uppercase) | Section headers, mission callouts |
| **Latin Technical** | `"Share Tech Mono", "Courier New", monospace` | Data readouts, coordinates, telemetry |
| **Body Propaganda** | `"Raleway", "Bebas Neue", sans-serif` | Body text, labels |
| **Fine Print** | `"Source Code Pro", monospace` | Sub-labels, micro data |

Always use **uppercase** for primary headings. Soviet constructivism demands it.

Mix Cyrillic transliteration sparingly for atmosphere: КОСМОС, ПРОГРЕСС, ВОСТОК.

## Layout Principles

### Constructivist Composition

Soviet Cosmism UI is **never symmetric and centered**. It is **dynamic and diagonal**.

```
┌──────────────────────────────────────────┐
│  ╱╱╱╱  MISSION HEADER (full width)  ╱╱╱╱ │
├──────────────┬───────────────────────────┤
│              │                           │
│  PRIMARY     │   WIDE DATA PANEL         │
│  EMBLEM      │   (2/3 width)             │
│  (1/3 width) │                           │
│              ├───────────┬───────────────┤
│              │  TELEMETRY│  STATUS       │
├──────────────┴───────────┴───────────────┤
│  ══════════ TICKER / SCROLLING DATA ════ │
└──────────────────────────────────────────┘
```

Key layout rules:

1. **Asymmetric split** — prefer 1/3 + 2/3 or 2/5 + 3/5 columns
2. **Diagonal accents** — use `transform: skewX(-8deg)` on headers
3. **Panel borders** — use `border-left: 3px solid <primary>` not full box borders
4. **Negative space** — let the void of space breathe around elements
5. **Layered depth** — background (stars) → midground (grid) → foreground (data)

## Animation Vocabulary

Every animation must carry **philosophical meaning** in Soviet Cosmism UI:

| Animation | Meaning | CSS/JS Implementation |
| --- | --- | --- |
| **Slow orbital rotation** | Eternal cosmic cycles | `rotate 120s linear infinite` |
| **Data transmission pulse** | Signal from cosmos to Earth | `scale 1.0→1.05 + opacity 0.6→1.0, 2s ease` |
| **Phosphor fade-in** | Consciousness emerging | `opacity 0→1, 1.5s ease, staggered` |
| **Scanline sweep** | Observation, surveillance, science | Moving gradient overlay |
| **Star parallax** | Depth of space, perspective shift | Three layers at 0.1x, 0.3x, 0.5x scroll |
| **Ticker crawl** | Continuous data from the noosphere | `translateX` marquee |
| **Gauge needle** | Instrument precision, control | SVG rotation from pivot point |
| **Grid flickering** | System processing thought | `opacity` random flicker on grid lines |

**Tempo guideline**: Soviet Cosmism UI moves *slowly and deliberately*. Nothing flinches. Minimum transition duration: 0.8s. Maximum animation speed: 60s orbital cycle.

## CSS Foundation

```
:root {
  /* Vostok palette (swap vars for other palettes) */
  --cosm-bg:       #0a0205;
  --cosm-panel:    #120810;
  --cosm-primary:  #cc1f1f;
  --cosm-accent:   #d4a017;
  --cosm-text:     #f0e8d8;

  --cosm-glow:     rgba(204,31,31,0.25);
  --cosm-grid:     rgba(204,31,31,0.15);

  /* Spacing rhythm */
  --cosm-unit:     8px;
  --cosm-border:   1px solid rgba(240,232,216,0.15);
  --cosm-accent-border: 3px solid var(--cosm-primary);
}

/* Base surface */
.cosm-surface {
  background: var(--cosm-bg);
  color: var(--cosm-text);
  font-family: "Share Tech Mono", monospace;
  position: relative;
  overflow: hidden;
}

/* Constructivist grid overlay */
.cosm-surface::before {
  content: '';
  position: absolute;
  inset: 0;
  background-image:
    linear-gradient(var(--cosm-grid) 1px, transparent 1px),
    linear-gradient(90deg, var(--cosm-grid) 1px, transparent 1px);
  background-size: 48px 48px;
  pointer-events: none;
  z-index: 0;
}

/* Film grain */
.cosm-surface::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("data:image/svg+xml,..."); /* noise SVG */
  opacity: 0.04;
  pointer-events: none;
  z-index: 1;

/* Panel component */
.cosm-panel {
  border-left: var(--cosm-accent-border);
  padding: calc(var(--cosm-unit) * 1.5) calc(var(--cosm-unit) * 2);

  position: relative;
  z-index: 2;
}

.cosm-skew {

  display: inline-block;
}

/* Glow text effect */
.cosm-glow {

    0 0 8px var(--cosm-primary),

  color: var(--cosm-data);
```

## What to AVOID

| ❌ Don't | ✅ Do Instead |
| --- | --- |
| Modern flat design (iOS/Material) | Aged, textured, panel-depth |
| Centered symmetric layouts | Asymmetric constructivist composition |
| Pastel or desaturated colors | Deep saturated primaries + black void |
| Fast snappy animations | Slow orbital, deliberate transitions |
| Western space aesthetic (NASA blue) | Soviet red/gold OR deep cosmic blue |
| Sans-serif lowercase body | Uppercase constructivist headers |
| Clean white backgrounds | Star-field black or near-black only |
| Generic sci-fi (Star Wars/Trek) | Specifically Soviet: utilitarian + utopian |

## Deliverable Checklist

Before presenting a Soviet Cosmism UI artifact, verify:

- Color palette is one of the three canonical sets (or a deliberate variant)
- At least one Cyrillic text element (КОСМОС, МИССИЯ, etc.)
- Background includes animated or static star field
- Grid overlay present at low opacity
- One orbital/rotation animation
- Typography uses monospace for data, uppercase for headers
- Layout is asymmetric (not centered grid)
- One Tsiolkovsky/Fedorov quote in ticker or header
- Film grain or noise texture applied
- All transitions ≥ 0.8s duration
