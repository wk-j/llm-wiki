# tinygrad

tinygrad คือ machine-learning framework / compiler stack ขนาดเล็กที่บทความ [[eternal-sloptember|The Eternal Sloptember]] ยกเป็นตัวอย่างงานจริงที่ผู้เขียนลองใช้ coding agents ช่วยทำ

## บทบาทในบทความ

ผู้เขียนบอกว่าได้เขียนบางส่วนของ tinygrad ด้วย agents แต่ยังรู้สึกว่าน่าจะทำเองได้ดีกว่าและเร็วกว่า. ตัวอย่างนี้ทำให้บทความไม่ใช่แค่ความเห็นลอย ๆ แต่เป็น field report จากงาน low-level/ML systems ที่มีรายละเอียดจริง

ได้อะไร: tinygrad เป็นเคสที่ทำให้ข้อวิจารณ์เรื่อง polish, correctness, และ process understanding มีน้ำหนักกว่า prototype web app ทั่วไป
