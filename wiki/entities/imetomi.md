# Imetomi

GitHub developer and creator of the [retro-futuristic-ui-design](https://github.com/Imetomi/retro-futuristic-ui-design) project — an open-source React + TypeScript component library implementing [[cassette-futurism]] CRT and LCD effects using pure CSS layering techniques. The project demonstrates that convincing retro display effects don't require WebGL or canvas.
