# Imetomi

นักพัฒนาบน GitHub และผู้สร้างโปรเจกต์ [retro-futuristic-ui-design](https://github.com/Imetomi/retro-futuristic-ui-design) — ซึ่งเป็น component library แบบ open-source ที่สร้างด้วย React + TypeScript เพื่อสร้างเอฟเฟกต์ CRT และ LCD แบบ [[cassette-futurism]] โดยใช้เทคนิคการซ้อน CSS ล้วนๆ โปรเจกต์นี้แสดงให้เห็นว่าเอฟเฟกต์จอภาพแบบเรโทรที่สมจริงนั้นไม่จำเป็นต้องใช้ WebGL หรือ canvas
