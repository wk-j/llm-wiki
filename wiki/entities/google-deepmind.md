# Google DeepMind / กูเกิล ดีปมายด์

Google DeepMind คือห้องวิจัย AI ของ Google ใน wiki นี้ ทีมโผล่อยู่ในสองเรื่องหลัก เรื่องแรกคือ **AlphaEvolve** ระบบที่ให้ AI ช่วยค้นวิธีแก้โจทย์วิทยาศาสตร์และคณิตศาสตร์ อีกเรื่องคือข้อเสนอของ [[demis-hassabis|Demis Hassabis]] ผู้ร่วมก่อตั้ง DeepMind ว่าควรตรวจ model ที่เสี่ยงสูงอย่างไรก่อนปล่อยให้คนใช้

สองเรื่องนี้เชื่อมกันตรง ๆ พอ AI ช่วยเร่งงานวิจัยได้มากขึ้น เราก็ต้องมีวิธีตรวจว่าความสามารถใหม่จะถูกใช้ผิดทางหรือหลุดจากการควบคุมหรือไม่

## AlphaEvolve ทำอะไร

AlphaEvolve เป็นระบบที่ให้ AI ลองเขียน code เพื่อแก้โจทย์ แล้วใช้โปรแกรมอีกชุดตรวจว่าคำตอบดีขึ้นจริงไหม วิธีทำงานมีห้าขั้น:

1. AI เสนอ code แบบหนึ่ง
2. ระบบนำ code นั้นไปทดสอบ
3. โปรแกรมให้คะแนนจากผลทดสอบ
4. ถ้าดีกว่าเดิมก็เก็บไว้
5. AI ใช้คำตอบที่ชนะเป็นจุดเริ่มของรอบต่อไป

จุดสำคัญคือคนไม่ต้องอ่านทุกคำตอบเอง เพราะโจทย์มีเกณฑ์ที่คอมพิวเตอร์ตรวจได้ทันที เช่น code รันเร็วขึ้นแค่ไหน หรือคำตอบคณิตศาสตร์ถูกหรือไม่

[[self-learning-for-agents-explained|แหล่งที่เล่าเรื่อง AlphaEvolve]] อ้างผลไว้สามอย่าง:

- ทำส่วนประมวลผล attention ให้เร็วขึ้น **32.5%**
- เจอวิธีคูณเมทริกซ์ที่ดีกว่าสถิติเดิมซึ่งอยู่มาตั้งแต่ปี **1969**
- ช่วยปรับขั้นตอนฝึก AI ให้เร็วขึ้น

ตัวอย่างสุดท้ายทำให้เห็นภาพง่ายที่สุด: AI ไม่ได้แค่ช่วยคนทำงาน แต่เริ่มช่วยปรับเครื่องมือที่ใช้สร้าง AI รุ่นต่อไปด้วย ใน [[three-learning-layers|กรอบการเรียนรู้สามชั้น]] นี่ต่างจากการจดบทเรียนไว้ใน memory เพราะตัวระบบกำลังช่วยปรับวิธีสร้าง AI รุ่นถัดไป

## AlphaEvolve ใช้ไม่ได้กับทุกงาน

วิธีนี้ต้องมีตัวตรวจที่ตอบได้ชัดว่า “ดีขึ้น” หรือ “แย่ลง” Code กับคณิตศาสตร์เหมาะ เพราะเครื่องรัน test และให้คะแนนได้

แต่งานอย่างตอบลูกค้า วางกลยุทธ์ หรือเขียนเรื่องให้น่าอ่าน ไม่มีคะแนนตายตัว ระบบจึงไม่รู้เองว่าคำตอบไหนดีกว่า ถ้าไม่มีคนช่วยตัดสิน

**ได้อะไร:** AlphaEvolve เก่งในงานที่ตรวจผลอัตโนมัติได้ ไม่ใช่หลักฐานว่า AI เรียนรู้เองได้ดีในทุกเรื่อง

## Demis Hassabis เสนอให้ตรวจ model ก่อนปล่อย

ใน [[framework-frontier-ai-dawning-new-age|A Framework for Frontier AI and the Dawning of a New Age]] Hassabis คาดว่า [[artificial-general-intelligence|Artificial General Intelligence หรือ AGI]] อาจมาในอีกไม่กี่ปี

AGI ในความหมายของเขาคือ AI ที่มีความสามารถทางความคิดกว้างใกล้มนุษย์ ไม่ได้เก่งแค่งานเดียว

เขาไม่ได้เสนอให้ปล่อยเทคโนโลยีวิ่งไปโดยไม่มีคนคุม กลับเสนอ [[frontier-ai-standards-body|หน่วยมาตรฐานสำหรับ Frontier AI]]

คำว่า Frontier AI ในที่นี้หมายถึง model ที่ความสามารถสูงจนเริ่มมีความเสี่ยงร้ายแรง หน่วยที่เสนอจะตรวจเรื่องอย่างการโจมตีระบบคอมพิวเตอร์ ความเสี่ยงด้านชีวภาพ การหลอกตัวตรวจ และการพยายามข้ามข้อห้าม ก่อนอนุญาตให้ปล่อย model

ตรงนี้ยังเป็น **ข้อเสนอ** หน่วยงานดังกล่าวยังไม่ได้ตั้งขึ้น และคำคาดการณ์ว่า AGI จะมาในอีกไม่กี่ปีก็ยังเป็นความเห็นของ Hassabis ไม่ใช่ข้อสรุปที่วงการเห็นตรงกัน

**ผลคือ:** DeepMind ปรากฏใน wiki ทั้งในฐานะทีมที่เร่งความสามารถของ AI และในฐานะต้นทางของข้อเสนอว่าควรสร้างกติกาให้ตามความสามารถนั้นทัน

## แยกให้ออกว่าอะไรคืออะไร

- ผลของ AlphaEvolve เป็นตัวเลขที่แหล่งข่าวอ้าง ต้องกลับไปดูงานวิจัยต้นทางก่อนใช้เป็นหลักฐานแข็ง
- AGI “อาจมาในอีกไม่กี่ปี” เป็นการคาดการณ์ของ Hassabis
- โลกที่ทรัพยากรเหลือเฟือเป็นภาพอนาคตที่เขาหวัง ไม่ใช่ผลลัพธ์ที่รับประกันแล้ว
- Frontier AI Standards Body เป็นแบบร่างของระบบกำกับ ไม่ใช่หน่วยงานที่มีอำนาจอยู่ตอนนี้

**ได้อะไร:** อ่านงานของ DeepMind โดยแยก “ผลงานที่ทดลองแล้ว”, “คำคาดการณ์” และ “ข้อเสนอเชิงนโยบาย” ออกจากกัน
