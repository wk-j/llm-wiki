# @Ctrl_Alt_Zaid / @Ctrl_Alt_Zaid

นักเขียนและนักวิจัยด้าน AI ที่โฟกัสเรื่องโครงสร้างพื้นฐานของ Agent Memory (ความจำของ Agent) และการสร้างระบบที่ช่วยให้ AI สามารถรักษาบริบท (context) ได้ในระยะยาว

เป็นผู้เขียนบทความวิเคราะห์แนวคิด [[llm-knowledge-bases|LLM Wiki]] ของ [[andrej-karpathy|Andrej Karpathy]] ในมุมมองของ autonomous agents (เอเจนต์ที่ทำงานด้วยตัวเอง) และเป็นหนึ่งในผู้ผลักดันโปรเจกต์ [[mercury|Mercury Agent]]

## ผลงานและแนวคิดสำคัญ
- **การเสนอแนวคิด [[hybrid-memory|Hybrid Memory Architecture]] (สถาปัตยกรรมความจำแบบผสม):** แยกส่วนที่คนอ่านกับส่วนที่เครื่องใช้คนละแบบกัน
- **การวิเคราะห์เรื่อง [[memory-drift|Memory Drift]] (ความจำเพี้ยน):** ปัญหาที่ Agent ยึดข้อมูลเก่าที่หมดอายุไปแล้วมาใช้งาน
- **Identity as human-owned, Memory as machine-efficient:** แนวคิดที่ว่า "ตัวตน" (Identity) ควรเป็นสิ่งที่มนุษย์ควบคุมได้ แต่ "ความจำ" (Memory) ควรเป็นสิ่งที่เครื่องจักรประมวลผลได้มีประสิทธิภาพ
