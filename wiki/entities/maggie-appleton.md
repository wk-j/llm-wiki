# Maggie Appleton / แม็กกี้ แอปเปิลตัน

**Maggie Appleton** เป็น Staff Research Engineer (และ Designer) ที่ [[github-next|GitHub Next]] เธอเชี่ยวชาญด้านการออกแบบระบบและเครื่องมือสำหรับ AI และเป็นผู้ผลักดันแนวคิด [[collaborative-ai-engineering|Collaborative AI Engineering]]

## บทบาทและแนวคิดหลัก

แม็กกี้มองว่าในยุคที่ AI สามารถเขียน code ได้ในราคาถูก ปัญหาของวิศวกรรมซอฟต์แวร์ได้ย้ายจาก "การสร้าง" (Implementation) ไปสู่ "การวางแผนและประสานงาน" (Alignment & Coordination)

- **Designer-Engineer Hybrid**: เธอเน้นย้ำว่าตำแหน่ง Designer และ Engineer เริ่มหลอมรวมกัน เพราะการออกแบบ UI และการเขียน code สำหรับ AI กลายเป็นเรื่องที่ต้องทำไปพร้อมๆ กัน
- **Alignment Advocate**: เธอเชื่อว่าเครื่องมืออย่าง GitHub, Slack หรือ Linear ในปัจจุบันยังไม่พร้อมรองรับโลกที่ Agent ทำงานเร็วมาก จนทีมตามไม่ทัน
- **ACE Vision**: เธอเป็นผู้นำในการพัฒนา [[ace]] ซึ่งเป็นสภาพแวดล้อมที่ทีมสามารถทำงานร่วมกับ Agent ได้อย่างใกล้ชิดและเห็นบริบท (context) เดียวกัน

## ผลงานสำคัญ

- **Collaborative AI Engineering Talk**: การบรรยายที่ชี้ให้เห็นว่า "การเพิ่มผลิตภาพรายบุคคล" (Individual Productivity) อาจทำลายผลิตภาพของทีมได้ถ้าขาดการประสานงานที่ดี
- **ACE (Agent Collaboration Environment)**: เครื่องมือต้นแบบที่เน้นการทำงานร่วมกันแบบ multiplayer ระหว่างคนและ AI
