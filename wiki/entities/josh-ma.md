# Josh Ma

**Josh Ma** คือผู้เขียนบล็อก [[what-weve-learned-building-cloud-agents|"What We've Learned Building Cloud Agents"]] (2026-06-02) ของ [[cursor|Cursor]] — สรุปบทเรียนหนึ่งปีของการสร้าง [[cloud-agents|cloud agent]].

ใจความที่เขาเสนอ: พอเอา agent ขึ้นคลาวด์ งานมันไม่ใช่การ port local agent ขึ้น server แต่กลายเป็นการสร้าง **operating layer** รอบตัว agent — [[agent-development-environment|environment]], [[durable-execution|durable execution]] (ผ่าน [[temporal|Temporal]]), การแยก conversation state, และแนวคิด [[self-healing-environments|self-healing environment]].
