# Comet Browser / เบราว์เซอร์ Comet

Comet คือ AI browser ของ [[perplexity|Perplexity]]. วิดีโอ [[how-perplexity-lost-ai-war|How Perplexity Lost the AI War]] มองว่านี่คือความพยายามออกจากการเป็น search tool ไปสู่การถือ interface และ distribution เอง

## ทำไม Perplexity ต้องทำ browser

logic ในวิดีโอคือ ถ้า Perplexity ยังเป็นแค่ destination หรือ app ผู้ใช้ต้องตั้งใจเปิดเข้ามาเอง. แต่ถ้า Perplexity ถือ browser ได้ มันจะเข้าใกล้จุดที่ [[google|Google]] ชนะ search เดิม: own interface, own default behavior, และ own user relationship

พูดง่าย ๆ คือ Comet ไม่ใช่แค่ feature. มันคือการแก้ [[distribution-moat|distribution moat]]

## ข้อจำกัด

วิดีโอเทียบกับ Microsoft Bing: Microsoft มี Windows, Bing placement, ChatGPT integration, GPT-4 access, และงบ marketing ใหญ่ แต่ยังเปลี่ยน search habit ได้ไม่มาก. ถ้าผู้เล่นระดับนั้นยังยาก Perplexity ก็ยิ่งเจอโจทย์หนัก

ผลคือ: browser/search เป็น habit ที่เหนียว. improvement ต้องชัดจนเปลี่ยนชีวิต ไม่ใช่แค่ดีกว่าเล็กน้อย

## Open questions

- Comet ทำ retention และ paid conversion ให้ Perplexity ดีขึ้นแค่ไหน
- ผู้ใช้ใช้ Comet เป็น daily browser จริง หรือใช้เป็น research surface เฉพาะงาน
- AI browser จะเป็น category ใหม่ หรือเป็น feature ที่ Chrome/Edge/Safari absorb ได้
