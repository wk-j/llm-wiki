# Trevin / เทรวิน

Trevin เป็น Software Engineer ที่ [[every-inc]] และเป็นผู้พัฒนาหลักของ [[compound-engineering]] (ce-skills) ซึ่งเป็นเครื่องมือที่ช่วยเพิ่มความแม่นยำและการทำงานร่วมกันระหว่างมนุษย์กับ AI agent

## ผลงานสำคัญ

- **Compound Engineering:** พัฒนาชุดคำสั่ง (skills) ที่เน้นเรื่องการทำ alignment และการลด "พิธีกรรม" (ceremony) ที่ไม่จำเป็นในการทำงานของ agent
- **[[playback-pattern|Playback Pattern]]:** ผู้นำเสนอวิธีการให้ agent "เล่นซ้ำ" (playback) สิ่งที่เข้าใจ (Stated, Inferred, Out-of-scope) เพื่อตรวจสอบความถูกต้องกับมนุษย์ก่อนเริ่มงานจริง

## แนวคิดในการทำงาน

Trevin เชื่อในการสร้าง agent ที่ **"Specifically wrong or specifically right"** (ผิดให้ชัดหรือถูกให้ชัด) เพื่อลดการคาดเดาแบบไม่มีหลักฐาน (hedging/speculation) และทำให้การแก้ไขทำได้รวดเร็ว (cheaper to correct)
