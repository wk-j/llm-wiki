# Project Glasswing

โครงการของ [[anthropic|Anthropic]] ที่เปิดตัวพร้อมกับ [[claude-mythos-preview|Claude Mythos Preview]] โดยมีเป้าหมาย 2 อย่าง: (1) ใช้ Mythos Preview เพื่อเสริมความปลอดภัยให้กับซอฟต์แวร์ที่สำคัญที่สุดของโลก และ (2) เตรียมความพร้อมให้อุตสาหกรรมสำหรับแนวปฏิบัติที่จำเป็นในการรับมือกับผู้โจมตีทางไซเบอร์ที่ใช้ AI

## สิ่งที่ทำ

- รัน [[claude-mythos-preview|Mythos Preview]] ผ่าน agentic scaffold (container + [[claude-code|Claude Code]]) เพื่อทดสอบซอฟต์แวร์ที่สำคัญ ทั้งแบบ open source และ closed source
- ประสานงานการเปิดเผยช่องโหว่ที่ค้นพบอย่างรับผิดชอบ (responsible disclosure) กับผู้ดูแล (maintainers) (ตามกรอบเวลา 90+45 วัน)
- ให้สิทธิ์การเข้าถึง Mythos Preview แก่ **กลุ่มพาร์ทเนอร์ในอุตสาหกรรมที่สำคัญและนักพัฒนา open source เพียงจำนวนจำกัด** — ไม่ได้เปิดให้ใช้เป็นการทั่วไป

## กระบวนการเปิดเผยข้อมูลอย่างมีการประสานงาน (Coordinated disclosure)

1.  Bug ทุกตัวจะถูกคัดกรอง (triage) โดยโมเดล
2.  Bug ที่มีความรุนแรงสูงสุดจะถูกส่งให้ผู้คัดกรองที่เป็นมนุษย์มืออาชีพเพื่อตรวจสอบความถูกต้อง
3.  รายงานที่ผ่านการตรวจสอบแล้วจะถูกเปิดเผยต่อผู้ดูแล
4.  ณ เวลาที่เผยแพร่บทความ มีช่องโหว่ที่ถูกแพตช์สมบูรณ์แล้วน้อยกว่า 1% ของที่ค้นพบทั้งหมด
5.  มีการเผยแพร่ SHA-3 commitments สำหรับช่องโหว่ที่ยังไม่ถูกแพตช์ เพื่อเป็นหลักฐานการค้นพบ (proof-of-possession)

## สเกล (ณ วันที่เผยแพร่)

- ค้นพบช่องโหว่ความรุนแรงระดับสูง/วิกฤตหลายพันรายการ
- การประเมินความรุนแรงโดยโมเดลและโดยมนุษย์ตรงกัน 89% (จากการตรวจสอบด้วยมือ 198 รายการ)
- ไม่พบ false positives เลยในการค้นหา memory corruption (โดยใช้ ASan เป็น oracle)
- มีการจ้างผู้รับเหมาด้านความปลอดภัยมืออาชีพเพื่อตรวจสอบด้วยตนเอง

## ท่าทีด้านการป้องกัน (Defensive posture)

เป้าหมายที่ประกาศไว้ของโครงการคือการช่วยให้ฝ่ายป้องกัน (defenders) สามารถเสริมความปลอดภัยของระบบที่สำคัญได้ *ก่อนที่* โมเดลที่มีความสามารถด้านไซเบอร์ระดับ Mythos-class จะกลายเป็นที่แพร่หลายจากผู้ให้บริการรายใดก็ตาม ในบล็อกระบุไว้อย่างชัดเจนว่า: "ในระยะสั้นนี้ [ผู้ที่ได้ประโยชน์] อาจเป็นผู้โจมตี หาก lab ที่พัฒนา frontier model ไม่ระมัดระวังเกี่ยวกับวิธีการปล่อยโมเดลเหล่านี้"

## แผนในอนาคต

- [[anthropic|Anthropic]] วางแผนที่จะเปิดตัวระบบป้องกันด้านไซเบอร์ใหม่พร้อมกับ "Claude Opus model ที่กำลังจะมาถึง" — เป็นการปรับปรุงระบบป้องกันบนโมเดลที่มีความเสี่ยงน้อยกว่า Mythos Preview
- มี **Cyber Verification Program** สำหรับผู้เชี่ยวชาญด้านความปลอดภัยที่การทำงานที่ถูกกฎหมายของพวกเขาได้รับผลกระทบจากระบบป้องกัน
- จะมีการเผยแพร่บทวิเคราะห์ทางเทคนิคโดยละเอียดของช่องโหว่ต่างๆ เมื่อได้รับการแพตช์แล้ว
