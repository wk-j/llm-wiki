# Rhys Sullivan / Rhys Sullivan

**Rhys Sullivan** ([@RhysSullivan](https://x.com/RhysSullivan)) เป็นผู้เขียนโพสต์ X [[i-dont-want-to-use-your-agent|i don't want to use your agent]]. ใน wiki นี้เขาสำคัญเพราะให้กรอบ product-side ของ [[bring-your-own-agent]]: บริษัทควรเปิด skills, docs, MCP/API/CLI และ product expertise ให้ agent ของผู้ใช้ ไม่ใช่บังคับให้ทุกคนใช้ in-app agent ของบริษัทอย่างเดียว.

## แนวคิดที่เกี่ยวข้อง

โพสต์ของเขาแยก user ออกเป็นสองกลุ่มชัด:

- user ทั่วไป ใช้ in-app chat แล้วได้ value เร็ว
- power user ใช้ daily-driver agent ของตัวเอง เพราะมี model แรง, local context, repo, terminal, และ workflow ที่ product agent เห็นไม่ครบ

ประเด็นนี้ต่อกับ [[coding-harness]] และ [[three-learning-layers]]: model/harness/context ไม่จำเป็นต้องอยู่ในมือบริษัทเดียว. บริษัทอาจเป็นเจ้าของ product context ส่วน user เป็นเจ้าของ harness กับ model.
