# Databricks / เดตาบริกส์

**Databricks** คือบริษัท data และ AI infrastructure ที่สร้าง Lakehouse Platform และผลิตภัณฑ์ agent เช่น Genie. บริษัทระบุว่าทีมวิศวกรกว่า 5,000 คนใช้ coding agent และได้สร้าง agent สำหรับลูกค้าหลายพันตัว.

ประสบการณ์นี้ทำให้ Databricks มองว่า harness แต่ละตัวกำลังกลายเป็น silo ใหม่. บริษัทจึงเปิด source [[omnigent|Omnigent]] ซึ่งเป็น [[meta-harness|ชั้นควบคุมเหนือ harness หลายตัว]] สำหรับประกอบ agent, บังคับ policy กลาง และแชร์ live session.
