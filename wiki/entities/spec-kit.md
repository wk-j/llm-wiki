# Spec Kit

**Spec Kit** คือ framework สำหรับ [[spec-driven-development|Spec-Driven Development]] จาก GitHub — ตัวใหญ่สุดของ movement ด้วยยอด ~90,000 GitHub stars ณ Q2 2026 (จาก ecosystem SDD รวมกันกว่า 200k stars ที่มี Kiro ของ Amazon, OpenSpec, BMAD ร่วมวง)

workflow หลักคือเขียน specification เป็นขั้น ๆ แล้วให้ coding agent สร้างโค้ดจาก spec จุดที่ถูกหยิบมาเป็นหลักฐานใน [[stop-writing-specs-start-writing-facts]]:

- ใน May 2026 Spec Kit ออกคำสั่ง `/speckit.spec-drift` — เครื่องมือจับอาการ **spec drift** (spec กับโค้ดหลุดจากกัน) ซึ่ง [[jaroslaw-wasowski|Wasowski]] อ่านว่าเป็นการยืนยันว่า drift เป็นความเสี่ยงจริงระดับต้องมี tooling เฉพาะ
- ตัว Wasowski เองหลังถอยจาก SDD ยังเก็บ **Spec Kit Step 1** ไว้ใช้เป็นกระดาษทด (sketchpad) — ส่วนเดียวที่รอดจากทุกโปรเจกต์
