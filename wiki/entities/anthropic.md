# Anthropic

AI safety lab and creator of the **[[claude|Claude]]** model family and **[[claude-code|Claude Code]]**. Self-described mission: build reliable, interpretable, and steerable AI systems.

## Products referenced across this wiki

- **[[claude|Claude]]** — the AI model family (Opus, Sonnet, Haiku)
- **[[claude-opus-4-7]]** — current flagship model (released 2026-04-16)
- **Claude Mythos Preview** — most capable Anthropic model as of 2026-04; release deliberately restricted while new cyber safeguards mature
- **[[claude-code|Claude Code]]** — terminal-based coding agent (also desktop, web, IDE extensions)
- **Claude API** (`platform.claude.com`) — developer API; supports effort levels, task budgets, advisor tool, MCP
- **Project Glasswing** — initiative addressing AI risks/benefits for cybersecurity

## Cyber strategy context

With Opus 4.7 Anthropic differentially reduced cyber capabilities during training and layered automatic safeguards on top. The stated strategy is to learn from 4.7's real-world deployment before broad release of **Mythos-class** models. Security professionals needing full cyber capability can apply to the **Cyber Verification Program**.
