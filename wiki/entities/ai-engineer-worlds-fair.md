# AI Engineer World's Fair

**AI Engineer World's Fair** คืองานรวมตัวของ AI engineer — โพสต์ [[piyalitt-codex-keynote-attention-not-token]] อ้างว่ามีผู้เข้าร่วมกว่า 7,000 คน. [[romain-huet|Romain Huet]] เปรียบกับงาน World's Fair ในอดีต: อนาคตถูกสร้างต่อหน้าสาธารณะ ไม่ได้ลอยมาจากที่ไหน

## Codex keynote (สรุปผ่าน Piyalitt)

Keynote นำโดย Romain Huet (Developer Experience, OpenAI) และ [[alexander-embiricos|Alexander Embiricos]] (Product, Codex) ก่อนส่งไม้ต่อ [[peter-steinberger|Peter Steinberger]]. ครอบคลุม:

- ทิศทางอาชีพ engineer ในยุค AI
- จังหวะปล่อย model และวิวัฒน์ coding experience
- Codex app, Goal Mode, Remote, Codex Cloud
- stack เปิดทุกชั้น (harness, AGENTS.md, App Server, plugins)
- [[value-maxing|value maxing]] และตระกูล GPT-5.6
- บทเรียน manager/worker loop และ [[attention-bottleneck|attention]] จาก Peter

## Fireside Chat ฝั่ง Claude Code

นอกจาก keynote ฝั่ง Codex งานนี้ยังมี Fireside Chat ที่ [[simon-willison|Simon Willison]] เป็นพิธีกร คุยกับ [[cat-wu|Cat Wu]] และ [[thariq-shihipar|Thariq Shihipar]] จากทีม [[claude-code|Claude Code]]. เคล็ดที่ Simon เก็บมาเขียนใน [[fables-judgement-simon-willison|Fable's judgement]] คือให้ [[fable|Fable]] ใช้ดุลพินิจเองแทนสั่งกฎตายตัว (ดู [[judgement-based-prompting]]).
