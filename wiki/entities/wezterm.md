# WezTerm

**WezTerm** คือ terminal emulator แบบ cross-platform ที่ config ด้วย Lua ได้. ใน [[l8-principals-agentic-engineering-workflow-kun-chen|วิดีโอ workflow ของ Kun Chen]] มันเป็นชั้นนอกสุดของ terminal-first setup ก่อนเข้า [[tmux]] และ [[neovim|Neovim]].

Kun ชอบ WezTerm เพราะใช้ได้ใกล้เคียงกันบน Mac, Windows, และ Linux และปรับแต่งด้วย Lua script ได้ละเอียด. จุดนี้สำคัญกับ workflow แบบ agent-heavy เพราะ terminal ไม่ใช่แค่หน้าต่าง shell แต่เป็น cockpit ที่ต้องวาง editor, shell, status, และ agent sessions หลายตัวให้อยู่ในมือเดียว.
