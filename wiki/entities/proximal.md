# Proximal

ทีมที่อยู่เบื้องหลัง benchmark [[frontierswe|FrontierSWE]] — benchmark วัด coding agent กับโจทย์ ultra-long-horizon ที่ยากที่สุด (เช่น optimize compiler, สร้าง optimizer สำหรับ ML training) โดยให้เวลา agent ถึง 20 ชั่วโมงต่อโจทย์

Proximal วิจารณ์ benchmark เดิมในแนวเดียวกับทีม [[datacurve|Datacurve]] คือ SWE-Bench Pro ยังเก็บโจทย์จาก pull request ขนาดเล็กถึงกลาง (เฉลยเฉลี่ย 107 บรรทัด) ซึ่งเล็กกว่างานจริงที่ใช้ agent ทำทุกวันนี้มาก
