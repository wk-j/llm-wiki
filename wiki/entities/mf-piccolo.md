# @mfpiccolo / เอ็ม-เอฟ-พิคโคโล

@mfpiccolo (คาดว่าเป็น Michael Piccolo) เป็นนักพัฒนาระบบและสถาปนิกซอฟต์แวร์ ผู้เป็นหัวหอกในการพัฒนาโปรเจกต์ [[iii-triple-i]] และเป็นเจ้าของบทความที่เป็นกระแสอย่าง [[the-harness-is-the-backend]]

## แนวคิดที่นำเสนอ

- **Harness Deconstruction:** Piccolo เชื่อว่าเราต้อง "รื้อสร้าง" (deconstruct) ความหมายของ Backend ใหม่เพื่อให้รองรับ AI Agent ได้อย่างแท้จริง
- **Complexity Collapse:** การยุบรวมหมวดหมู่เครื่องมือที่กระจัดกระจายให้เหลือเพียง Primitives ที่เรียบง่าย
- **Stochastic Debugging:** การระบุปัญหาเรื่องความยากในการไล่สายการทำงานเมื่อระบบ Agent และ Backend แยกจากกัน และเสนอทางออกด้วย Unified Traceability
