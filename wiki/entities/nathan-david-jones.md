# Nathan David Jones

Concept artist whose retro-futuristic UI work is a primary visual reference for the [[cassette-futurism]] aesthetic in web/game design. His pieces feature amber-on-black CRT palettes, chunky bezels, interactive text prompts, and phosphor glow effects — the visual language that Imetomi's [[imetomi-retro-futuristic-ui|retro-futuristic-ui-design]] React components recreate.
