# Nathan David Jones

Concept artist ที่ผลงาน UI แนว retro-futuristic ของเขาเป็นภาพอ้างอิงหลักสำหรับสุนทรียศาสตร์แบบ [[cassette-futurism]] ในการออกแบบเว็บ/เกม ผลงานของเขามีลักษณะเด่นคือจานสีแบบจอ CRT สีอำพันบนพื้นดำ, ขอบจอหนา, interactive text prompts, และเอฟเฟกต์แสงฟอสเฟอร์ — ซึ่งเป็นภาษาสายตาที่ React component ของ Imetomi ในโปรเจกต์ [[imetomi-retro-futuristic-ui|retro-futuristic-ui-design]] นำมาสร้างใหม่
