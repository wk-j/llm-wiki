# Leo Robinovitch

Independent developer known for building [[tui|TUI]] applications and tooling in Go. Blogs at [theleo.zone](https://theleo.zone).

## Projects

| Project | Description |
|---|---|
| [viewport](https://github.com/robinovitch61/viewport) | Reusable Go component: scrollable, searchable, ANSI/Unicode-aware text viewport for Bubble Tea TUIs |
| [lore](https://github.com/robinovitch61/lore/) | [[terminal-pager]] built on the viewport component; Leo's daily-driver replacement for `less` |
| [kl](https://github.com/robinovitch61/kl) | Kubernetes log viewer TUI: multi-cluster, multi-namespace, searchable, JSON prettification |
| [wander](https://github.com/robinovitch61/wander/) | HashiCorp Nomad TUI |

The viewport component is the shared core across these applications, factored out as a standalone reusable library.
