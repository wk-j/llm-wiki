# Lavish

**Lavish** คือเครื่องมือ/skill ของ [[kun-chen|Kun Chen]] สำหรับ planning ผ่าน interactive HTML artifact. ใน [[l8-principals-agentic-engineering-workflow-kun-chen|วิดีโอ workflow]] Lavish แทน wall-of-text plan ใน terminal ด้วยหน้า visual ที่แสดง options, prototype, annotation, และ decision points.

จุดสำคัญคือ artifact ใช้ design system ของ project จริง ทำให้ feedback อยู่ใกล้ product มากกว่า markdown plan. คนสามารถ comment เฉพาะส่วนที่ไม่ชอบหรือเลือกตัวเลือกในหน้า artifact ได้ แล้วส่งกลับ agent โดยไม่ต้องอธิบายใหม่ยาว ๆ.

Lavish จึงอยู่ตรงรอยต่อของ [[html-artifacts|HTML artifacts]], [[agent-experience|Agent Experience]], และ [[agentic-engineering|Agentic Engineering]]: มันทำให้ช่วง planning ชัดขึ้นก่อนปล่อย agent ไป implement ยาว ๆ.
