# GitClear

**GitClear** คือบริษัท/เครื่องมือวิเคราะห์ developer productivity จาก git activity. ใน [[agentic-code-review|Agentic Code Review]] ของ [[addy-osmani|Addy Osmani]] GitClear เป็นแหล่งข้อมูลที่ทำให้เห็นช่องว่างระหว่าง raw code output กับ delivered value.

Addy อ้างงานของ GitClear ว่า daily AI users ผลิต raw output ประมาณ 4 เท่าของ non-users แต่เมื่อเทียบกับ output ของตัวเองปีก่อน productivity gain จริงอยู่ราว 12%. Bill Harding จาก GitClear ยังยอมรับว่า 12% นี้มี selection bias บางส่วน เพราะ developer ที่แข็งแรงกว่าอาจอยู่ในกลุ่ม AI users มากกว่า.

## ที่ทางในวิกินี้

GitClear ช่วยให้เห็นว่า "โค้ดมากขึ้น" ไม่เท่ากับ "value มากขึ้น". ช่องว่างนี้คือพื้นที่ของ [[agentic-code-review|agentic code review]], [[orchestration-tax|orchestration tax]], และ [[acceptance-bottleneck|acceptance bottleneck]].
