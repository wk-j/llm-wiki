# Hugging Face

AI platform และ community hub สำหรับ models, datasets, และ demos ในบริบท [[introducing-genebench-pro|Introducing GeneBench-Pro]] ของ [[openai|OpenAI]] เป็นที่ที่ OpenAI เปิดชุดตัวอย่าง 10 คำถามของ [[genebench-pro|GeneBench-Pro]]

## บทบาทใน GeneBench-Pro

OpenAI ไม่ได้ open-source benchmark ทั้งหมด แต่เผยแพร่ public package บน Hugging Face เพื่อให้คน inspect ตัวอย่างโจทย์, data files, และ case-study package ได้

ผลคือ Hugging Face ทำหน้าที่เป็น distribution surface สำหรับ benchmark sample ไม่ใช่ผู้สร้าง benchmark
