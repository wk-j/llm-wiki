# The Singularity Is Nearer

ชื่อ author/blog ที่มากับบทความ [[eternal-sloptember|The Eternal Sloptember]] บน `geohot.github.io`. ใน wiki นี้ใช้ชื่อนี้ตาม metadata ที่ผู้ใช้ให้มา และยังไม่แยกยืนยันตัวตนผู้เขียนนอกเหนือจากตัวบทความ

## จุดยืนในบทความนี้

บทความนี้เป็นเสียงคัดค้าน coding agents ที่แรงกว่าสาย [[agentic-coding-trap|Agentic Coding is a Trap]] และ [[building-pi-world-of-slop|Building pi in a World of Slop]] ตรงที่ไม่ใช่แค่เตือนเรื่อง review หรือ harness แต่ตั้ง claim ว่า agents ในแนวปัจจุบันยัง program ไม่ได้จริง

แกนคือ output ของ agent ดูเหมือน programming มากขึ้น แต่ไม่ได้แปลว่ามี process แบบ programmer อยู่ข้างหลัง
