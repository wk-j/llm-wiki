# Faros AI

**Faros AI** คือบริษัทด้าน engineering intelligence / developer productivity metrics. ใน [[agentic-code-review|Agentic Code Review]] ของ [[addy-osmani|Addy Osmani]] Faros เป็นแหล่งข้อมูลหลักที่ชี้ว่า AI adoption เพิ่ม throughput แต่ดันภาระ review และ defect ขึ้นพร้อมกัน.

Addy อ้างข้อมูล March 2026 จาก 22,000 developers / 4,000 teams: code churn เพิ่ม 861%, incidents-to-PR ratio เพิ่ม 242.7%, defect rate ต่อ developer เพิ่มจาก 9% เป็น 54%, median review duration เพิ่ม 441.5%, และ PR ที่ merge โดย zero review เพิ่ม 31.3%.

ต้องอ่านด้วย caveat: Faros ขาย product ในตลาดนี้ จึงไม่ใช่ผู้สังเกตการณ์ที่ไม่มีผลประโยชน์. แต่ตัวเลขถูกใช้เป็นสัญญาณว่าความเร็วฝั่ง production วิ่งนำ capacity ฝั่ง review ไปไกล.
