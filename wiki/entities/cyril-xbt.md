# @cyrilXBT

Solo builder และ content creator บน X ที่เขียนเกี่ยวกับเครื่องมือ AI, การเทรด crypto, และ workflow การทำงานให้มีประสิทธิภาพสูง (productivity) เขาให้นิยามกลุ่มเป้าหมายของตัวเองว่าเป็น solo builder "ระดับกลางถึงสูง" ที่ไม่ต้องการการสอนแบบจับมือทำ

เขาถูกอ้างอิงใน wiki นี้ในฐานะผู้เขียนคู่มือการจัดโครงสร้างไฟล์ [[claude-md|CLAUDE.md]] (21 เม.ย. 2026) ซึ่งได้ตกผลึก template 7 ส่วนขึ้นมา: Project Overview / Tech Stack / Coding Conventions / **Never Do This** / File Structure / Current Sprint Goals / Important Context แนวทางของเขานี้มีความขัดแย้งเชิงสร้างสรรค์ (productive tension) กับแนวคิดเรื่อง [[instruction-budget]] ของ [[alex-ker|Alex Ker]] — ทั้งคู่เห็นตรงกันว่า prompt ควรเขียนโดยมนุษย์และมีความเฉพาะเจาะจง แต่ไม่เห็นด้วยเรื่องปริมาณ

นอกจากนี้ยังมีข้อมูลว่าเขาทำ dashboard สำหรับ bot แจ้งสัญญาณ crypto (ใช้ Next.js + Tailwind + Supabase, Binance API) เป็นสินค้า επί πληρωμή บน Gumroad และเขายังใช้ CLAUDE.md สำหรับโปรเจกต์เขียน content ด้วย
