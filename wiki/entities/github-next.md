# GitHub Next / กิทฮับ เน็กซ์

**GitHub Next** เป็นทีมวิจัยและพัฒนา (R&D) ภายใน [[github|GitHub]] ที่เน้นการสร้างเครื่องมือและเทคโนโลยีใหม่ๆ ที่อาจจะมาเปลี่ยนวิถีการสร้างซอฟต์แวร์ในอนาคต (Department of "F*** Around and Find Out")

## โปรเจกต์ที่น่าสนใจ

- **GitHub Copilot**: จุดเริ่มต้นของการนำ LLM มาช่วยเขียน code
- **ACE (Agent Collaboration Environment)**: สภาพแวดล้อมการพัฒนาซอฟต์แวร์แบบ Multiplayer ที่เน้นการใช้ Agent ในระดับทีม
- **Copilot Workspace**: การเปลี่ยนจากการคุยกับ AI เป็นการทำงานร่วมกับ AI ในระดับแผนงาน (Planning)

## ปรัชญาการทำงาน

ทีมเน้นการสร้าง "Research Prototype" เพื่อสำรวจว่า primitives ของการพัฒนาซอฟต์แวร์ในอนาคตควรเป็นอย่างไร โดยมองข้ามข้อจำกัดของเครื่องมือในปัจจุบันอย่าง Pull Request (PR) หรือ Issue
