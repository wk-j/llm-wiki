# Sourcegraph

บริษัท developer tools ที่ก่อตั้งปี 2013 โดย Quinn Slack และ Beyang Liu เริ่มจาก "Google สำหรับ source code" — สร้าง code search engine ระดับ enterprise ที่ index repository ขนาดใหญ่ได้

ในยุค AI, Sourcegraph เคยมีผลิตภัณฑ์ Cody ซึ่งเป็น AI coding assistant ที่ผูกกับ code search ของตัวเอง ปัจจุบัน flagship ของบริษัทคือ [[amp|Amp]] — coding agent รุ่นใหม่ที่ออกแบบให้ทำงานแบบ agent-first ไม่ใช่ assistant ข้างจอ

## ตำแหน่งในตลาด AI coding agents

Amp ของ Sourcegraph แข่งในตลาดเดียวกับ:

- [[claude-code|Claude Code]] (Anthropic)
- [[openai-codex|OpenAI Codex]] / Codex CLI
- [[opencode|OpenCode]]
- [[thclaws|thClaws]]
- [[kimi-k2-6|Kimi]]
- Cursor

จุดต่างของ Amp คือมาจากบริษัทที่ทำ code search ระดับ enterprise มาก่อน — มี asset เรื่อง indexing, retrieval, และ scale ของ codebase ใหญ่ที่ไม่ใช่ทุกคู่แข่งจะมี
