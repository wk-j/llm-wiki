# Ryan Lopopolo / ไรอัน โลโปโปโล

Member of Technical Staff ที่ [[openai]] ผู้นำเสนอแนวคิด [[harness-engineering]] และการสร้างซอฟต์แวร์โดยใช้ Agent ล้วน

## บทบาทและแนวคิด
- **Agent-First Workflow**: Ryan สั่งห้ามทีมเขียนโค้ดเองใน editor แต่ต้องสั่งผ่าน Agent เท่านั้น เพื่อฝึกทักษะการสั่งงานและวางระบบ
- **Token Billionaire**: เชื่อว่าทางไปสู่ AGI คือการใช้โมเดลทำงานแทนมนุษย์ในปริมาณมหาศาล (เขามีปริมาณการใช้ output token มากกว่า 1 พันล้านโทเคนต่อวัน)
- **Harness Engineering**: เน้นการสร้าง "บังเหียน" (Harness) รอบตัว Agent เพื่อควบคุมคุณภาพและลดการแทรกแซงจากมนุษย์

## โปรเจกต์ที่เกี่ยวข้อง
- **[[openai-codex]]**: Ryan ทำงานในทีมที่พัฒนาเครื่องมือสำหรับ coding agent ภายใน OpenAI
- **Harness Infrastructure**: พัฒนาระบบ review และ linting อัตโนมัติที่ช่วยให้ Agent สามารถ self-heal ได้
