# Claude Web / คลอดด์ เว็บ

**Claude Web** คือตัวอินเตอร์เฟซหลักของ Claude ผ่านเบราว์เซอร์ (claude.ai) ซึ่งถือเป็น [[ai-agent|AI Agent]] รูปแบบหนึ่งที่ใช้ [[claude|Claude]] model

ในนิยามของ [[matt-pocock|Matt Pocock]], Claude Web คือ **Agent** ที่เกิดจาก:
- **Model**: [[claude-opus-4-7|Claude Opus]] หรือโมเดลอื่นในตระกูล Claude
- **Harness**: เว็บอินเตอร์เฟซ, ระบบจัดการ session บนคลาวด์, Artifacts, และชุดเครื่องมือที่จำกัด (เช่น การอัปโหลดไฟล์/วิเคราะห์ข้อมูล)
- **Environment**: สภาพแวดล้อมระบบคลาวด์ของ Anthropic

Claude Web แตกต่างจาก [[claude-code|Claude Code]] แม้จะใช้โมเดลเดียวกัน เพราะมี **Harness** และ **Environment** ที่ต่างกัน โดย Claude Web เน้นการโต้ตอบผ่าน UI และการสร้าง Artifacts ในขณะที่ Claude Code เน้นการเข้าถึงไฟล์ในเครื่องและการรันคำสั่ง terminal
