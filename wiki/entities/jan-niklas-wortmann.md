# Jan-Niklas Wortmann / ยาน-นิคลาส วอร์ตมันน์

**Jan-Niklas Wortmann** เป็นผู้สัมภาษณ์และ creator ด้าน software engineering / AI coding. ใน source [[code-isnt-free-mario-zechner-hard-truths-coding-ai|Code Isn't Free]] เขาคุยกับ [[mario-zechner|Mario Zechner]] เรื่อง [[pi-agent|pi]], workflow ของ coding agent, [[spec-driven-development|Spec-Driven Development]], local AI, และภาระของ open source maintainer ในยุค [[clanker-slop|clanker slop]].

บทบาทของเขาในคลิปนี้คือดึงความตึงระหว่างสองฝั่งออกมา: agent ทำให้การสำรวจ solution เร็วขึ้นจริง แต่ถ้าคนไม่อ่าน ไม่คิด และไม่รับผิดชอบผลลัพธ์ ความเร็วจะกลายเป็นหนี้.
