# Datacurve

ทีม/องค์กรที่อยู่เบื้องหลัง benchmark [[deepswe|DeepSWE]] — benchmark วัด AI coding agent กับงาน long-horizon ที่ออกแบบให้ปลอดการปนเปื้อนและตรวจด้วย verifier ที่อิงพฤติกรรม

จุดยืนหลักที่ทีมประกาศคือ benchmark สาธารณะเดิมทำให้ model ดูสูสีกันเกินจริง พอเจองานจริงกลับแยกห่างเป็นช่องว่างกว้างที่ตรงกับสิ่งที่ developer สัมผัสได้ทุกวัน — DeepSWE จึงถูกออกแบบมาเพื่อทำให้ช่องว่างนั้นปรากฏเป็นตัวเลข
