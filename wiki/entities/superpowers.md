# superpowers

**Type:** Agent skill repository (จาก Obra) — [[matt-pocock|Matt Pocock]] เรียกว่า "น่าจะเป็น skills repo ที่ดังที่สุดที่มีอยู่"

superpowers ถูกยกขึ้นมาในพอดแคสต์ Matt Pocock × [[david-ondrej|David Ondrej]] (2026-06-19) ในฐานะ **ขั้วตรงข้ามเชิงปรัชญา** ของ skill repo ของ Matt:

- **Matt** ชอบ skill แบบ **procedures** — user เป็นคนคุม ขับเอง (ดู [[skill-procedures-vs-abilities]])
- **superpowers** เลือกทางที่ **model เป็นคนคุม** (หยิบ ability เอง) มากกว่า

ไม่ใช่ว่าใครผิด — แค่คนละแนวคิดเรื่อง "ใครควรถือพวงมาลัย" ระหว่างคนกับ model. Matt ยกตัวอย่างว่าถ้าใครคิดถึงฟีเจอร์ brainstorming จาก superpowers ก็เอากลับมาใช้ได้ หลังจากลองเริ่มจาก blank slate แล้ว
