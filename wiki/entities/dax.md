# Dax (OpenCode)

**Dax** เป็นผู้สร้าง [[opencode]] (Open-source coding agent)

Dax เป็นตัวอย่างของนักพัฒนาที่ "การเขียนโค้ดคือการวางแผน" (Coding === Planning) เขาอธิบายว่าเวลาเจอเรื่องยากๆ เขาไม่ได้เริ่มต้นจากการเขียน spec ยาวๆ เพื่อสั่ง AI (Spec Driven Development) แต่การพิมพ์โค้ด การกำหนด type การลองจัด folder structure คือกระบวนการคิดและหาคำตอบว่าฟีเจอร์นั้นควรทำงานอย่างไร ซึ่งเป็นธรรมชาติของโปรแกรมเมอร์ส่วนใหญ่ การพยายามบังคับให้คนเหล่านี้หยุดเขียนโค้ดและไปนั่งเขียนแต่ prompt จึงเป็นการขัดขวางกระบวนการคิดตามธรรมชาติ
