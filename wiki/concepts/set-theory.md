# Set Theory / ทฤษฎีเซต

Set theory คือสาขาคณิตศาสตร์ที่ศึกษาชุดของสิ่งต่าง ๆ และความสัมพันธ์ระหว่างชุดเหล่านั้น. ในวิดีโอของ [[mlhf|MLHF]] set theory เป็นพื้นหลังของเรื่อง [[infinity|infinity]], [[cardinality|cardinality]], และ [[zfc-set-theory|ZFC]]

## ทำไมเกี่ยวกับ infinity

ถ้าจะถามว่า infinity มีหลายขนาดไหม เราต้องถามผ่านเซต:

- เซตของจำนวนนับ
- เซตของเลขคู่
- เซตของจำนวนจริง

แล้วใช้ cardinality เปรียบเทียบขนาดของเซตเหล่านั้น
