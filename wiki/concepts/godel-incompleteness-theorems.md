# Gödel's Incompleteness Theorems / ทฤษฎีบทความไม่สมบูรณ์ของ Gödel

Gödel's incompleteness theorems คือผลลัพธ์ใน mathematical logic ที่บอกว่าระบบ formal ที่แรงพอจะทำ arithmetic ได้ มีข้อจำกัดเชิงโครงสร้างของมันเอง

## ใจความแบบไม่ลง technical

ถ้าระบบ axiom แรงพอและไม่ขัดแย้งในตัวเอง ระบบนั้นจะไม่ complete. จะมี statement บางอย่างที่จริง แต่พิสูจน์ไม่ได้ภายในระบบนั้น

นี่ต่างจากการที่เรายังหาบทพิสูจน์ไม่เจอ. ประเด็นคือไม่มีบทพิสูจน์ในระบบนั้นเลย

## ทำไมสะเทือนคณิตศาสตร์

ก่อนหน้านี้เราอาจนึกว่าถ้าเลือก axiom ดีพอ คณิตศาสตร์จะกลายเป็นระบบที่พิสูจน์ความจริงทุกอย่างได้. Gödel ทำให้ภาพนั้นพัง: ต่อให้แก้ระบบให้ตอบโจทย์หนึ่งได้ ก็ยังมีโจทย์ชนิดใหม่ที่ระบบตอบไม่ได้อยู่ดี
