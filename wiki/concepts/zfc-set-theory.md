# ZFC Set Theory / ทฤษฎีเซต ZFC

ZFC คือระบบ axiom ของ set theory ที่ใช้เป็นรากฐานของคณิตศาสตร์สมัยใหม่จำนวนมาก. ชื่อย่อมาจาก Zermelo-Fraenkel set theory with Choice

## Axiom คืออะไร

Axiom คือข้อเริ่มต้นที่ระบบยอมรับไว้ก่อน แล้วใช้พิสูจน์ข้ออื่นต่อ. วิดีโอของ [[mlhf|MLHF]] ยกตัวอย่างจากเรขาคณิตแบบ Euclid เพื่อให้เห็นว่าแต่ละสาขาคณิตศาสตร์ตั้งอยู่บนข้อเริ่มต้นบางชุด

ZFC เป็นชุด axiom ที่ระดับลึกกว่า เพราะใช้เป็นฐานให้คณิตศาสตร์หลายแขนง

## ข้อจำกัด

ZFC ตอบคำถามได้มาก แต่ไม่ตอบได้ทุกอย่าง. [[continuum-hypothesis|Continuum Hypothesis]] เป็นตัวอย่างสำคัญ: ถ้า ZFC ไม่ขัดแย้งในตัวเอง ก็พิสูจน์ไม่ได้และหักล้างไม่ได้จาก ZFC
