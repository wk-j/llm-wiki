# Memory Scoring / การให้คะแนนความจำ

การเพิ่มข้อมูลกำกับ (Metadata) ให้กับความจำแต่ละชิ้นในระบบของ Agent เพื่อใช้ในการจัดลำดับความสำคัญ (Ranking) และการตัดสินใจเลือกข้อมูลมาใช้งานได้อย่างแม่นยำ

## เกณฑ์การให้คะแนน

ระบบ Agent Memory ที่มีประสิทธิภาพมักใช้เกณฑ์เหล่านี้:
1. **Freshness (ความสดใหม่):** ข้อมูลล่าสุดควรได้รับคะแนนสูงกว่า เพื่อป้องกันปัญหา [[memory-drift|Memory Drift]]
2. **Confidence (ความมั่นใจ):** ข้อมูลที่ได้รับการยืนยันหลายครั้ง หรือมาจากแหล่งที่เชื่อถือได้จะมีคะแนนสูงกว่าข้อมูลที่ Agent สรุปเอาเอง
3. **Importance (ความสำคัญ):** ข้อมูลที่เป็นหลักการพื้นฐานหรือความชอบหลักของผู้ใช้ (Core preferences) จะมีคะแนนสูงถาวร
4. **Reinforcement (การตอกย้ำ):** ข้อมูลที่ถูกเรียกใช้บ่อยหรือถูกผู้ใช้ยืนยันซ้ำจะได้รับการเพิ่มคะแนนโดยอัตโนมัติ

## ทำไมถึงสำคัญ?

หากไม่มีการให้คะแนน ข้อมูลทุกอย่างจะ "ตะโกน" ใส่ Agent ด้วยความดังที่เท่ากันทั้งหมด (Everything competes equally) ทำให้ Agent แยกแยะไม่ได้ว่าควรจะเชื่อข้อมูลชิ้นไหนดีเมื่อเกิดความขัดแย้ง การให้คะแนนช่วยให้ Agent มีวิจารณญาณในการเลือกข้อมูลที่ "ดีที่สุด" มาใช้งาน
