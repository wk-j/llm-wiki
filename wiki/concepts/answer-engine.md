# Answer Engine / เครื่องมือตอบคำถาม

Answer engine คือ search experience ที่พยายามตอบคำถามให้เสร็จในหน้าเดียว แทนที่จะส่งผู้ใช้ไปไล่เปิด link เอง. [[perplexity|Perplexity]] ทำให้ pattern นี้เด่นขึ้นด้วยคำตอบพร้อม citation

## ต่างจาก search engine แบบเดิมอย่างไร

search engine แบบเดิมให้รายการหน้าเว็บ. ผู้ใช้ต้องอ่าน, เทียบ source, และสรุปเอง. Answer engine พยายามย้ายงานนั้นเข้ามาใน product:

- ค้นหา source
- อ่านและ synthesize
- ตอบเป็นภาษาธรรมชาติ
- แสดง citation ให้ตรวจได้

ได้อะไร: ภาระของผู้ใช้ลดลง โดยเฉพาะงาน research ที่ถามเป็นคำถามยาว ๆ

## จุดเปราะ

ข้อได้เปรียบนี้ copy ได้เร็วถ้า player ใหญ่มี model, index, และ user surface อยู่แล้ว. วิดีโอ [[how-perplexity-lost-ai-war|Mondo Startups]] จึงบอกว่า “answer with sources” จาก differentiation ของ Perplexity กลายเป็น table stakes ของตลาด AI search

อีกจุดเปราะคือ trust. ถ้า answer engine สรุปเนื้อหาจาก publisher แล้วผิด, ใกล้เคียงเกินไป, หรือให้ attribution ไม่ถูก ความไว้ใจของผู้ใช้และเจ้าของ content จะสั่น
