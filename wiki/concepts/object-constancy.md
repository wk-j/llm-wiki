# Object Constancy / การคงภาพของคนทั้งดีและไม่ดีไว้พร้อมกัน

**Object constancy** ใน psychoanalytic/object relations context คือความสามารถในการรู้ว่าคนคนเดียวกันมีทั้งด้านดีและด้านไม่ดีได้พร้อมกัน. บทความ [[defence-mechanism|Defence Mechanism]] ใช้มันอธิบายว่าคนที่ healthy กว่าจะทนภาพผสมแบบนี้ได้.

## ทำไมเรื่องนี้สำคัญ

ถ้าคนมี object constancy เขายังรักษาภาพของอีกฝ่ายไว้ได้ แม้กำลังโกรธ ผิดหวัง หรือเจ็บ. เขาอาจคิดว่า "คนนี้ทำร้ายความรู้สึกฉันตอนนี้" โดยไม่ต้องสรุปว่า "คนนี้เลวทั้งหมด".

ถ้าคนรวมภาพนี้ไม่ได้ ใจอาจใช้ splitting: แยกภาพเป็นดีหมดหรือแย่หมด. วันนี้รักมาก พรุ่งนี้เกลียดหมด. วันนี้เห็นตัวเองเก่งมาก พอพลาดครั้งเดียวก็เห็นตัวเองไร้ค่า.

**ได้อะไร:** object constancy ทำให้ความสัมพันธ์รับ conflict ได้โดยไม่พังเป็นขาว/ดำทันที.

## เชื่อมกับ defense mechanisms

หน้า [[defense-mechanism-hierarchy|Defense Mechanism Hierarchy]] โยง object constancy กับ defense แบบ splitting-based ในงานของ [[otto-kernberg|Otto F. Kernberg]]. Kernberg มองว่า borderline personality organization เกิดเมื่อเด็กไม่สามารถ integrate helpful และ harmful mental objects เข้าด้วยกันได้.

ในภาษาง่าย ๆ: ถ้าภาพของคนอื่นหรือภาพตัวเองยังรวมกันไม่ติด ใจต้องใช้ defense ที่แยกภาพออกจากกันเพื่อกัน anxiety.

**ผลคือ:** object constancy ไม่ใช่แค่ "มองโลกบวก". มันคือความสามารถในการถือความจริงสองด้านพร้อมกัน.

## ข้อควรระวัง

ใน wiki นี้ควรใช้ object constancy เป็น concept เพื่ออ่านความสัมพันธ์และ defense pattern ไม่ใช่ใช้ diagnosis คนอื่นจากพฤติกรรมไม่กี่ครั้ง. ความสัมพันธ์จริงมีบริบทเรื่อง trauma, power, safety, และ evidence ที่ต้องดูแยก.
