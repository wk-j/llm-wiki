# Soviet Cosmism UI Style

Source: Claude conversation describing Soviet Cosmism as a UI design language.

## Philosophical foundation

Rooted in the ideas of Nikolai Fedorov, Konstantin Tsiolkovsky, and Vladimir Vernadsky — the belief that humanity's destiny is literal conquest of the cosmos: immortality, resurrection, interplanetary civilization. The UI style inherits **revolutionary optimism**, **collectivist scale**, and **technological mysticism**.

## Visual DNA

- **Constructivist geometry** — bold diagonals, triangles, circles, radiating lines from a central point. Layouts feel like propaganda posters translated into interface.
- **Futurism meets folk** — industrial machine forms (gears, orbits, Sputnik spheres) combined with Slavic ornamental patterns and grain textures.
- **Color palette** — Cosmist red (`#C1272D`), golden ochre, Sputnik silver, midnight blue, rocket-exhaust orange-white. Black is structural, never decorative.
- **Typography** — Cyrillic-inspired geometric letterforms even in Latin text. Wide, spaced, all-caps. No serifs. Text as monumental statement.
- **Symbology** — atoms, orbits, cosmonaut silhouettes, launchpads, solar system diagrams, the upward-reaching hand. Icons feel embossed on metal.
- **Texture** — aged propaganda-poster grain, linoleum-print texture, lithographic dot patterns. Never photographic.
- **Layout** — asymmetric but balanced. Grids broken by a single diagonal thrust (rocket trajectory, radial beam). Everything has implied momentum.

## UI translation

- Dashboards as **mission control panels** — data displayed as gauge clusters, not cards
- Navigation uses **radial or orbital metaphors** — hubs and spokes, not sidebars
- Buttons feel **stamped, not clicked** — raised metal aesthetic, blocky corners
- Progress indicators use **orbit animations** or countdown aesthetics
- Borders are functional, structural, telegraph-relay thick

## Key tension

Simultaneously **utopian warmth** (humanity united, reaching for stars) and **cold machine logic** (everything is a system, a calculation, a five-year plan). Good Soviet Cosmism UI holds both at once.
