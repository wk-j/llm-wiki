# This New Protocol Will Change AI Coding Forever (ACP)

Video by Zed Industries introducing the **Agent Client Protocol (ACP)**, an open standard for communication between AI agents and code editors.

## Key Takeaways

- **Agent Client Protocol (ACP):** A universal standard (inspired by LSP and MCP) that allows AI agents to interact with IDEs regardless of the specific agent or editor.
- **Solving Fragmentation:** Currently, AI tools require custom integrations for every editor. ACP acts as a "universal adapter."
- **Live, Interactive Coding:** Demonstrates real-time code proposals inside the Zed editor with line-by-line accept/reject functionality.
- **Multi-Agent Flexibility:** Users can swap AI "brains" (e.g., Claude, Gemini, custom agents) while keeping the same UI/UX.
- **Built on JSON-RPC:** Uses a structured, proven messaging format similar to MCP and LSP.
- **Open Invitation:** Zed encourages other editors (VS Code, JetBrains) to adopt the standard to prevent editor lock-in.

## Payoff

AI becomes a first-class citizen of the development environment—capable of planning, executing, and validating changes through a standardized interface rather than just a side-chat.
