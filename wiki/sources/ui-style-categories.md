# UI Style Categories

A comprehensive taxonomy of UI/UX design styles, compiled from a Claude conversation. Covers both industry-standard categories and more niche/emerging aesthetics.

## Key taxonomy (11 categories, ~40+ styles)

| Category | Styles |
|---|---|
| **Flat & Material** | Flat Design, Material Design, Material You, Semi-Flat / Flat 2.0 |
| **Skeuomorphic & Realistic** | Skeuomorphism, [[neumorphism]], [[glassmorphism]], [[claymorphism]] |
| **Minimalist** | Minimalism, Ultra-minimalism / Zen UI, Swiss Style, Typographic UI |
| **Dark & Atmospheric** | Dark Mode, [[cyberpunk-neon]], [[vaporwave]], [[synthwave]] |
| **Sci-Fi / Futuristic** | [[diegetic-ui]], Holographic UI, [[brutalist-futurism]], [[retro-futurism]] |
| **Retro & Vintage** | [[brutalism]], Retro / 8-bit, [[y2k-aesthetic]], [[memphis-design]], [[art-deco]] |
| **Organic & Natural** | [[blobmorphism]], Biomorphic Design, Nature-inspired, Cottagecore UI |
| **Bold & Expressive** | Maximalism, Editorial / Magazine UI, Anti-design, Experimental Typography |
| **3D & Spatial** | 3D UI, visionOS Style, AR UI, Isometric UI |
| **Corporate & Enterprise** | Fluent Design (Microsoft), Apple HIG, Carbon Design (IBM), Ant Design |
| **Gradient & Color-driven** | Gradient Mesh, Aurora UI, Duotone, Monochromatic |

## Sci-Fi sub-styles (detailed)

- **Diegetic HUD** — Iron Man / JARVIS. Cyan on dark, arc meters, scan lines, bracket targeting.
- **Holographic UI** — Minority Report. Semi-transparent panels, floating particles, projected-light feel.
- **Brutalist Futurism** — Raw data, heavy type, sharp borders, red accents. Confrontational.
- **Retro-Futurism** — 80s sci-fi aesthetic. CRT scan lines, pixel fonts, neon pink/purple/green.

## Color associations

- Diegetic HUD: Cyan `#00d4ff`, Amber `#ef9f27`, Red (alert)
- Neural/Bio-tech: Purple `#a78bfa` + Mint
- Retro-Futurism: Neon pink, purple, green
